# CIFAR-10 Image Classifier 🖼️

A CNN-based image classifier built with PyTorch achieving ~85% accuracy on the CIFAR-10 dataset.

## Tech Stack
Python · PyTorch · NumPy · Matplotlib · Scikit-learn

## Results
| Metric | Score |
|--------|-------|
| Test Accuracy | ~85% |
| F1 Score | 0.85 |
| Framework | PyTorch |
| Dataset | CIFAR-10 (60K images, 10 classes) |

## Setup & Run
pip install -r requirements.txt
python main.py
